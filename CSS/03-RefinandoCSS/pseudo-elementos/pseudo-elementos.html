Pseudo-elementos

Usado para adicionar elementos extras na página sem a necessidade de declara-lo no HTML, ou para selecionar uma parte de algum elemento.

identificador por ::paalavra-chave

Pseudo-elemento- ::first-letter

Seleciona a primeira letra da primeira linha

Propriedades disponiveis:

  • color
  • background
  • padding
  • border
  • font
  • Propriedades de texto

Pseudo-elemento- ::first-line

Seleciona a primeria linha de um texto

Propriedades disponiveis:

  • color
  • background
  • font
  • Propriedades de texto

Estiliza o plano de fundo de uma tela cheia como dialog ou video. quando a tela do video fica cheia voce pode mudar o fundo preto pra outra cor

Pseudo elemento- ::marker

Permite estilizar os topicos ou numeros de ula lista (li) ou entao a flrcha de um elemento summary

Propriedades aceitars:
  • color
  • content
  • white-space
  • font
  • animation
  • transition
